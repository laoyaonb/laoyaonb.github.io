关于

建站目的

社团要求

精弘网络技术部要求新部员搭建个人博客以用于学习和交流。
欢迎大家来精弘网络参观！

个人愿景

在搭建维护博客的过程中，学会博客的基本应用，提高视野。
同时将自己的学习笔记和心得记录于此，用于学习和交流。

lyn’s blog 中的内容

后端学习

使用Go语言(golang)进行主要开发
博客中将会上传学习的经历和笔记，以及遇到的一些 独特的坑 问题的解决方案。

C/C++

博主并没有将C和C++分开学习，因此可能在C++中混有大量一些C语言的习惯

C++

C++主要用于学习竞赛算法
博客里会不定期更新一些有思维含量的题 (也可能是博主脑袋短路，半天没想到方法的题)
题目来源主要以洛谷和codeforces（俄罗斯的一个全球性的刷题以及比赛网站，题目以英文为主）为主。

C语言

由于博主的专业开设C语言学科，因此不得不将C语言的语法单独拎出来用于应付考试。
博客内容主要以知识点解释汇集以及备考为主。

Python

博主最早学习的编程语言是浙江高考要求的Python.
所以也是学习最久，使用最熟练的语言。
但是由于学习过程中较少使用，所以博客内容应当较少，且内容主要往Python的优势方面发展(爬虫，人工智能…)
等到2024年上半年专业开设Python课程时，会与C语言更新方向相似。

因此会更新从零开始的MATLAB的学习经历和笔记，以及一些实战的建模题目。

总结
